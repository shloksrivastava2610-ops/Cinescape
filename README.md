# CineTale - Movie and Series Discovery App

A web application to discover, search, explore and save your favourite movies and TV series.
Built as part of a JavaScript + API Integration college project.

## API Used
TMDB API (The Movie Database) - https://www.themoviedb.org/documentation/api
Free and well documented.

## Features
- Browse trending movies and TV series
- Search by title with debouncing
- Filter by genre
- Sort by rating, release date or alphabetically
- Save favourites via localStorage
- Infinite scroll for continuous browsing
- Loading indicators while fetching data
- Fully responsive UI (mobile, tablet, desktop)

## Technologies
- HTML, CSS, Vanilla JavaScript
- TMDB REST API (via fetch)
- localStorage for saving favourites
- Array HOFs - filter, map, sort, find, some
