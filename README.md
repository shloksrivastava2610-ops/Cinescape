CineVerse — Movie & Series Discovery App

A web application to discover, search, and explore movies and TV series.
Built as part of a JavaScript + API Integration college project.

## API Used
[TMDB API](https://www.themoviedb.org/documentation/api) — The Movie Database, free and well documented

##  Planned Features
- Browse trending movies and TV series
- Search by title (with debouncing)
- Filter by genre
- Sort by rating or release date
- Save favourites via localStorage
- Dark / Light mode toggle
- Fully responsive UI (mobile, tablet, desktop)
- Loading indicators while fetching

##  Technologies
- HTML, CSS, Vanilla JavaScript
- TMDB REST API (via fetch)
- localStorage for saving favourites
